## ⚠️ 重要更正

### 1. FPE_A 定義改變

**之前（錯誤）**：
- ❌ 公司官方財報公布的 Forward P/E
- ❌ 從財報狗/buffet code 直接提取

**現在（正確）**：
- ✅ **用「公司財報指引中自己給的財測/來期業績」去推 forward EPS → forward P/E**
- ✅ 從公司官方財報中的財測/來期業績數據推導
- ✅ 數據來源：財報狗/buffet code 的財報搜尋結果（提取財測/來期業績）

**推導方式**：
```
FPE_A = 當前股價 / 公司財測/來期業績的 Forward EPS
```

### 3. FPE 的使用與地位（重要）

#### FPE 在系統中的定位

**1. FPE 在 P2 的安全性分析中權重不高**
- FPE 雖然僅是預測，在基本面安全性分析權重不高
- 但在股價漲幅潛力面卻是非常重要的因素

**2. FPE 更像是 P3 的因子**
- FPE 與 P2 的主題安全性分析有點偏離
- FPE 更像是 **P3 判斷現在股價是否便宜/合理/昂貴的因子之一**
- 舊文案有定案：這些數據都要繼承給 P3 作為綜合判斷調整 cat 使用

**3. FPE 的使用限制**
- ⚠️ **禁止使用 FPE 單一項來決定**
- ✅ FPE A/B 僅是眾多決策因子的一環

**4. FPE_A vs FPE_B 的特性**
- **FPE_A**：
  - 公司公布的 FPE_A 公布時就已經股價反應
  - **沒有未來增幅的效用**
  
- **FPE_B**：
  - 真正對股價有推動力的是**後來不斷更新的 FPE_B**
  - 但是問題：不夠準確+來源無法驗證+可能人為操縱
  - 所以外來資訊僅做為參考與輔助
  - **我們的系統，才是真正能全方位算出最接近未來 FPE 的工具**

---

### 4. Forward P/E 的使用注意事項

#### Forward P/E 的風險

**預估來自哪裡？**
- Forward P/E 的關鍵是 EPS 預估值
- 來源通常來自法人機構或公司財測
- 若預估失準，估值也會偏差

**不適合高度不確定產業**
- 例如新創公司、週期股
- 未來 EPS 波動大，Forward P/E 可能誤導投資決策

**別單看倍數高低下判斷**
- 一家 Forward P/E 20 倍的公司，可能成長性遠勝另一家 10 倍的公司
- 重點在合理性，不是絕對值

#### Forward P/E 的實際應用

**與自身歷史比**：
- 觀察一家公司過去 3～5 年的 Forward P/E 區間
- 了解目前估值相對是高還是低

**與同業比較**：
- 評估同產業中不同企業的 Forward P/E 水準
- 找出估值偏低且成長性不差的標的

**搭配 PEG（本益成長比）使用**：
- 將 Forward P/E 除以未來獲利成長率
- 可更客觀衡量合理性

## 📋 需要調整的項目

### 1. FPE_A 的提取方式

**新方式**：
- 從財報狗/buffet code 搜尋結果中提取「公司財測/來期業績」
- 提取 Forward EPS（基於公司財測）
- 計算 FPE_A = 當前股價 / Forward EPS

**實作**：
- 需要實作從 CSE 搜尋結果中提取財測/來期業績的邏輯
- 需要獲取當前股價
- 需要計算 FPE_A

**選項 1：Yahoo Finance**（目前使用）
- 優點：跨市場覆蓋廣、多分析師共識
- 缺點：非官方、可能被限流
