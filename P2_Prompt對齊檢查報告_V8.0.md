### 4. P2-5 安全性 Gate（三級制）描述不完整

**現有 Prompt 有提到，但描述不夠詳細：**
- ⚠️ 有提到 PASS/PARTIAL/FAIL，但沒有明確說明：
  - ❌ PASS：盈利穩定且成長、現金流健康、財務指標優於同業 → 進 Phase 3
  - ❌ WATCH：盈利波動但可承受、現金流尚可、個別指標弱於同業但非致命 → 進 Phase 3（後段控倉）
  - ❌ FAIL：虧損且無轉機跡象、現金流惡化、債務壓力大 → 停止於 Phase 2

**⚠️ 注意：現有 Prompt 使用的是 PASS/PARTIAL/FAIL，但任務精神提到的是 PASS/WATCH/FAIL，需要確認是否統一。**
