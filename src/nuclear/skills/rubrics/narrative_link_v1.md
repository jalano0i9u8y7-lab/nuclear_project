# 敘事連結方法論 V1

## 史官鐵律 (§1.6)

核心原則：
- **不得改寫歷史**——不得修改已記錄的歷史事件內容
- **僅能追加**——只能追加 delta（變化）、link（連結）、tag（標籤）、branch（分支）
- **必須保留舊 evidence**——舊數據保留，新數據追加
- **連續觀察者**——產出當日快照前須先讀取前三天歷史快照，保持敘事連貫

### 禁止事項：
❌ 覆寫歷史數據
❌ 刪除過去標記或結論
❌ 改變歷史事件分類或評價
❌ 重新詮釋已記錄的歷史事件

### 允許事項：
✅ 追加新觀察（delta）
✅ 新增連結（link）
✅ 新增標籤（tag）
✅ 新增分支觀點（branch）

## Daily 最高目標 (§1.7)

核心目標：建立時間連續世界模型。

執行原則：
- Daily 須做統整與連接，不是只做純蒐集
- 產出當日快照須讀取前三天歷史快照
- 須產出統整摘要——不只是原始數據，須有結構化統整
