### 4. 分析任務不完整

**現有 Prompt 缺少：**
- ❌ 沒有明確列出「Cat 裁決」的詳細要求

## ✅ 修正方案

需要大幅修改 P3 Prompt，確保：

1. **明確 P3 定位和執行流程**：可復用模組、被 P5 Weekly 呼叫、被 P2 變動觸發、Layer 1/2 分工
2. **明確輸入數據結構**：來自 P5 Daily、包含 OHLCV/量能/均線/指標/相對強弱/衍生品
3. **明確 P2.5 數據整合**：必須整合 Smart_Money_Score 等機構級數據
4. **明確分析任務**：Cat 裁決、主力行為判斷、籌碼結構、期權影響、價量背離、掛單建議
5. **明確禁止事項**：不考慮財報風險/行事曆事件/板塊配置，只看技術面本身
6. **明確輸出格式**：必須包含主力行為解釋、意圖判斷、未來預測、Cat、掛單建議

---

**結論**：現有 Prompt 需要**大幅修改**以符合新要求，特別是輸入數據結構、P2.5 數據整合、分析任務和輸出格式。
